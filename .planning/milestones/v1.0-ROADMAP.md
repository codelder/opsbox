## Milestone Summary

**Key Decisions:**

- Decision: 先收紧后添加 — 避免在松散基础上叠加新测试
- Decision: NDJSON mock 格式 — useStreamReader 期望 { type: 'result', data: {...} }
- Decision: XSS 断言分开检查 — highlight() 将关键词包在 <mark> 标签中
- Decision: A11Y-03 分两次导航 — 避免 mock route 冲突
- Decision: 用 waitForFunction 替代 waitForTimeout — 事件驱动而非固定延迟

**Issues Resolved:**

- Fixed mock response formats to match actual backend API contracts (LLM: `{ backends: [...] }`, S3: `{ profiles: [...] }`)
- Fixed glob route pattern intercepting Vite source files
- Fixed `waitForLoadState('networkidle')` timeouts due to persistent backend connections
- Fixed XSS assertion for highlighted content (mark tags split escaped entities)

**Issues Deferred:**

- None — all planned work completed

**Technical Debt Incurred:**

- None
